# DJ Slammer App

An interactive DJ learning app for young aspiring DJs, built for Tanner (DJ Slammer).

## Current Status

**Phase 1: Vision Quiz** - Live now!
- Interactive questionnaire to gather requirements from Tanner
- Mobile-friendly design
- Built to understand what features matter most

## Planned Features

Based on Tanner's vision:
- Daily DJ style selector
- Interactive DJ controller interface
- Guided tutorials ("tells you what to press")
- Beat-making and music creation tools
- DJ equipment education

## Tech Stack

- Pure HTML/CSS/JavaScript (for now)
- Will expand based on Tanner's feedback

## Development

This app is being built collaboratively with input from an 8-year-old DJ who's already performing shows!
